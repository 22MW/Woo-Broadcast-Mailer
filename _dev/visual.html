La release 2.0.2 está publicada en GitHub Releases con tag v2.0.2 y asset woo-broadcast-mailer.zip.

El ZIP de release incluye solo archivos runtime y excluye documentación interna, Node, fuentes de desarrollo, cachés y archivos locales.

  • Prueba del updater en staging.

Hecho

  • Audiencia global multi-fuente con deduplicación.
  • Preview de destinatarios únicos.
  • Envíos instantáneos y programados con Action Scheduler.

Base comercial

Herramienta para enviar comunicaciones segmentadas desde WooCommerce usando compradores, roles, listas Mail Mint o emails manuales, con preview, programación, logs y edición de textos de emails.

Vista local interna. No incluir en ZIP, release ni deploy público.
