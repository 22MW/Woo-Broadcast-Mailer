01 / Broadcast Mailer

Para enviar mensajes sencillos a listas, compradores y grupos concretos.

Esta es la parte de emailing. Sirve cuando necesitas avisar a personas concretas desde WooCommerce sin exportar clientes ni crear una campaña completa en otra herramienta.

Qué problema resuelve

Muchas veces solo necesitas mandar un aviso: un cambio de fecha, una actualización de producto, una instrucción importante o un mensaje operativo. El problema es reunir bien a los destinatarios sin copiar y pegar listas de un lado a otro.

01

Seleccionas destinatarios

Compradores de productos WooCommerce, roles, emails manuales, Broadcast Lists propias y las listas que ya tienes en tu WordPress desde Mail Mint, AcyMailing, FluentCRM, MailPoet o Newsletter.

02

Combinas fuentes

Puedes mezclar varias fuentes en un solo envío y el plugin deduplica emails para no escribir dos veces a la misma persona.

03

Revisas antes de enviar

La vista previa muestra destinatarios únicos. Si alguien no debe recibir ese aviso, puedes quitarlo solo para ese envío.

04

Preparas el mensaje

Escribes el body, insertas imagen, botón, bloque destacado o separador, y puedes enviar HTML sin plantilla global si no quieres wrappers externos.

05

Guardas para reutilizar

Las Broadcast Lists guardan destinatarios. Las plantillas guardan solo el body del mensaje, sin asunto ni configuración de envío.

06

Envías o programas

Puedes enviar ahora o programar por lotes para controlar mejor el ritmo y revisar logs después.

Casos reales de Broadcast

Situaciones normales donde esta parte tiene sentido.

Aviso a compradores

Vendiste un curso, producto o evento y necesitas avisar solo a quienes lo compraron.

Lista recurrente

Tienes un grupo al que escribes a menudo. Lo guardas como Broadcast List y lo reutilizas.

Listas que ya existen en tu WP

Ya tienes listas creadas en Mail Mint, AcyMailing, FluentCRM, MailPoet o Newsletter, y quieres seleccionarlas directamente como fuente del envío.

Un cambio, una incidencia, una instrucción o un recordatorio que no necesita una campaña de marketing.

Mensaje visual simple

Necesitas una imagen, un botón y una caja destacada, pero no un constructor de newsletters.

Body reutilizable

Guardas una estructura de mensaje y la vuelves a cargar cuando necesitas mandar algo parecido.

02 / Email String Editor

Para adaptar textos genéricos de WooCommerce y sus plugins a tu negocio.

Esta parte no sirve para enviar un broadcast. Sirve para controlar frases que ya aparecen en emails automáticos de WooCommerce o plugins relacionados.

Qué problema resuelve

WooCommerce y sus extensiones suelen traer textos genéricos. No siempre están mal, pero muchas veces no encajan con el tono, idioma o forma de comunicar de tu negocio. El editor permite localizarlos y adaptarlos sin tocar plantillas.

03 / FAQs

Preguntas básicas antes de entender el plugin.

¿Es una newsletter?

No. Es para avisos puntuales desde WooCommerce. No sustituye una plataforma de marketing.

¿Puedo usar listas que ya tengo?

Sí. Puedes seleccionar las listas disponibles en tu WordPress desde plugins como Mail Mint, AcyMailing, FluentCRM, MailPoet o Newsletter, además de Broadcast Lists propias y emails manuales.

¿Puedo guardar listas propias?

Sí. Las Broadcast Lists permiten guardar grupos de emails para reutilizarlos.

¿Puedo guardar mensajes?

Sí. Las plantillas guardan solo el body del mensaje, no la audiencia ni la programación.

¿Diseña emails completos?

No es un constructor complejo, pero permite crear un diseño sencillo con imagen, botón, bloque destacado y separador, y guardar ese body como plantilla para reutilizarlo.

¿Qué hace el editor de strings?

Permite adaptar textos genéricos de emails WooCommerce y plugins relacionados, incluso por idioma.
